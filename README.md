# FALI — Fantasy Availability & Limitation Index

A FAIR-Sport–inspired, decision-first injury & availability tool for NBA fantasy.

## What it does
- Builds a weekly **Availability Score (0–100)**
- Flags **DNP risk** and **minutes limitation risk**
- Provides **attribution** of main limitation drivers
- Outputs **start / bench** guidance

This is a PAI-derived tool focused on injury & short-term availability.

pip install -r requirements.txt
